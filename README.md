# GuessTheMovie

Angular web + mobile application. [Docs](https://docs.nativescript.org/angular/code-sharing/intro).

## Running the web app

```
npm i
npm start
```

## NativeScript

Follow [this guide](https://docs.nativescript.org/start/quick-setup) to set up NativeScript.

### Running the Android app
```
npm i
npm run android
```

### Running the iOS app
```
npm i
npm run ios
```
